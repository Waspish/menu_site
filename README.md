# 🍔 Menu Site

Сайт меню ресторана быстрого питания — одностраничный статический сайт с акциями и категориями блюд.

## 🚀 Запуск

Скачайте репозиторий и откройте `index.html` в любом браузере.

```bash
git clone https://github.com/Waspish/menu_site.git
cd menu_site
```

## 📝 Особенности

- **Статический HTML/CSS** – не требует сборки или сервера

## 🛠 Используемые технологии

- HTML
- CSS
